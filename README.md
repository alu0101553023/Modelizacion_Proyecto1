Modelos de Regresión en Desmos
Este repositorio contiene modelos interactivos creados con Desmos para visualizar diferentes tipos de regresión. Cada modelo incluye un diagrama de dispersión de datos reales, su función de regresión ajustada y elementos visuales que facilitan la interpretación de los resultados.

Código de colores estandarizado
Para facilitar la interpretación de todos los modelos, hemos establecido un código de colores consistente:

Morado: Diagrama de dispersión (puntos del dataset original)

Verde: Función de regresión (modelo ajustado a los datos)

Naranja: Interpolaciones (puntos estimados dentro del rango de los datos)

Azul: Residuos (distancias verticales entre datos reales y predicciones)

Rojo: Proyección 2030 (valor esperado en la intersección con x = 2030)

Visualización típica

En cada modelo encontrarás:

Los datos originales en morado

La curva/modelo ajustado en verde

Puntos de interpolación en naranja (valores estimados entre años conocidos)

Segmentos verticales en azul representando los residuos

Un punto rojo destacado que muestra la proyección para el año 2030

Cómo usar
Abre cualquiera de los archivos .html en tu navegador

Serás redirigido a Desmos con el modelo precargado

Puedes interactuar con los deslizadores, hacer zoom y explorar los datos

Observa cómo los residuos (azules) cambian según el modelo seleccionado

Notas
Los modelos utilizan datos reales o simulados según el contexto

Los parámetros de regresión están optimizados para minimizar los residuos

La proyección al 2030 es una estimación basada en la tendencia del modelo
